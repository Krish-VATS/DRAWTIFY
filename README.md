# 🎨 Drawtify Whiteboard

**Drawtify** is a high-performance, minimalist whiteboard application built for creators who value focus. Based on the Excalidraw engine, it has been stripped of all distracting branding and heavy UI elements to provide a "subtle" and professional drawing experience.

[![Live Demo](https://img.shields.io/badge/Launch-Drawtify-6366f1?style=for-the-badge&logo=vercel)](https://drawtify-with-nextjs.vercel.app)

---

## ✨ Key Customizations

- **Subtle UI:** Removed high-contrast borders and heavy purple branding for a modern "floating" aesthetic.
- **Clean Workspace:** Stripped away "Help," "Sign Up," and "Excalidraw Plus" promotion banners.
- **Custom Branding:** Replaced all original icons and favicons with a minimalist "D" identity.
- **Optimized for Vercel:** Fully configured for high-speed global deployment.
- **Live Collaboration:** Real-time syncing enabled via custom environment variables.
